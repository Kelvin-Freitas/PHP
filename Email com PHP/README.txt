Aqui é mostrado de forma básica, como se envia um email através do php, utilizando o gmail.
É importante falar, que foi utilizado o Composer, uma ferramenta para geerenciamento de dependências e com ele também foi utilizado a biblioteca PHPMailer, para enviar os emails com segurança e facilidade.
Para testar, será necessário que utilize o composer mas também é possível realizando o download da biblioteca e gerenciando as utilizações manualmente.
Lembre-se de alterar os dados no arquivo Config.php e se utilizar sem o composer retirar o require do autoload.
